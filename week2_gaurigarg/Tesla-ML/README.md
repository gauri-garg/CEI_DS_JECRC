## Overview
This project focuses on analyzing the Tesla deliveries dataset from 2015 to 2025. The current stage includes data understanding and exploration.

## Files
- `01_data_understand.ipynb`: Jupyter notebook containing the initial exploratory data analysis (EDA).
- `tesla_deliveries_dataset_2015_2025.csv`: The dataset containing Tesla vehicle delivery records.

## Instructions
To view and run the analysis:
1. Ensure you have the necessary Python data science libraries installed (e.g., `pandas`, `matplotlib`, `seaborn`).
2. Open the notebook in one of the following environments:
   - Jupyter Notebook
   - JupyterLab
   - VS Code
   - Google Colab
3. Run the notebook cells sequentially to reproduce the data exploration steps.
